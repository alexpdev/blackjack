blackJack API Documentation

Class Hierarchy

  • blackJack.Deck.Card - Card Object == contents of Deck Object.
  • blackJack.Game.Driver - Responsible for putting pieces together and driving them forward.
  • blackJack.Players.Player - Object representing a player playing against the dealer.
    • blackJack.Players.Dealer - Dealer Object. Controls most aspects of the game.
  • Exception
    • blackJack.Deck.DeckEmpty - Deck Empty when no more cards in deck to deal.
    • blackJack.Deck.InvalidType - Exception Class for Invalid type comparison.
  • list
    • blackJack.Deck.Deck - Deck of cards.
  • PyQt6.QtWidgets.QDialog
    • blackJack.MenuBar.Settings - Open new window with editable options that effect gameplay.
  • PyQt6.QtWidgets.QDialogButtonBox
    • blackJack.MenuBar.About - Dialog with information about the Program.
  • PyQt6.QtWidgets.QGroupBox
    • blackJack.PlayerBox.PlayerBox - PlayerBox Subclass of QGroupBox.
  • PyQt6.QtWidgets.QLabel
    • blackJack.PlayerBox.CardWidget - Store the image of the card it represents for GUI display.
  • PyQt6.QtWidgets.QMainWindow
    • blackJack.Window.Window - Window MainWindow for Blackjack UI.
  • PyQt6.QtWidgets.QMenuBar
    • blackJack.MenuBar.MenuBar - QMenuBar instance for main window menubar.
  • PyQt6.QtWidgets.QMessageBox
    • blackJack.Window.BrokeDialog - BrokeDialog box to show player if their score is over 21.
  • PyQt6.QtWidgets.QPushButton
    • blackJack.Window.HitButton - Hit Button triggered by player wants to be dealt another card.
    • blackJack.Window.NewGameButton - New Game Button.
    • blackJack.Window.StandButton - Stand Button is for players who want their turn to be over.
